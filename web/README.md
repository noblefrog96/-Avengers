# 연습용 팀 소개 사이트 (Vite)

`춘식팀.com`처럼 **구조가 단순한 1페이지 랜딩**을 연습하기 위한 템플릿입니다.  
원본 사이트의 문구/이미지/자산을 그대로 복제하지 않고, **레이아웃/섹션 구조 위주**로 새로 구성했습니다.

## 구성

- 상단 고정 헤더 + 모바일 메뉴 토글
- 히어로(CTA + 요약 스탯)
- 소개 / 멤버 / 프로젝트 / 문의 섹션
- 푸터(연도 자동 표기)

## 실행

```bash
cd web
npm install
npm run dev
```

## 수정 포인트

- `index.html`: 섹션/문구/링크 수정
- `src/style.css`: 색/타이포/카드/레이아웃 수정
- `src/main.js`: 메뉴 토글/헤더 스크롤 효과 등

## 이미지 교체(권장 워크플로)

제가 외부 사이트에서 이미지를 “그대로 복제”해 가져오는 건 불가해서, **교체용 슬롯**을 만들어뒀습니다.

- `web/public/images/hero.svg` → 원하는 히어로 이미지로 교체(파일명 유지 추천)
- `web/public/images/member-a.svg` ~ `member-d.svg` → 멤버 이미지로 교체

파일명만 유지하면, 코드 수정 없이 이미지가 바로 바뀝니다.
