# 연습용 팀 소개 사이트 (Vite)

`춘식팀.com`처럼 **구조가 단순한 1페이지 랜딩**을 연습하기 위한 템플릿입니다.  
원본 사이트의 문구/이미지/자산을 그대로 복제하지 않고, **레이아웃/섹션 구조 위주**로 새로 구성했습니다.

## 구성

- 상단 고정 헤더 + 모바일 메뉴 토글
- 히어로(CTA + 요약 스탯)
- 소개 / 멤버 / 프로젝트 / 문의 섹션
- 푸터(연도 자동 표기)

## 실행

```bash
cd web
npm install
npm run dev
```

## 수정 포인트

- `index.html`: 섹션/문구/링크 수정
- `src/style.css`: 색/타이포/카드/레이아웃 수정
- `src/main.js`: 메뉴 토글/헤더 스크롤 효과 등
